# audit_Census_Income_Classifier_v03.md

## Цель
Финальная проверка готовности проекта *Census Income Classifier v02* к публикации на GitHub и перечень обязательных шагов доводки.

---

## Что исправить перед пушем (по приоритету)

### A. Убрать тяжёлые артефакты из Git-истории
- Переместить модели и артефакты в `releases/` или выгружать по запросу.
- Альтернатива: использовать Git LFS для `data/models/*.joblib`, `data/artifacts/*`, крупных PNG.
- В `README` добавить ссылку на релиз с артефактами.
- В `.gitignore` исключить `data/artifacts/` и `data/models/` целиком, либо хранить только CSV/JSON с метриками.

### B. Очистить ноутбуки
- Выполнить `jupyter nbconvert --clear-output --inplace notebooks/**/*.ipynb`.
- Сохранить «чистые» версии без выводов, чтобы не тащить пути и промежуточные результаты в историю.

### C. Сееды
- В 01/02/03 ноутбуках зафиксировать `np.random.seed(…)` и `random_state=…` во всех моделях, разбиениях и примерах.
- Проверить детерминизм кросс-валидации и визуализаций.

### D. README — мелкие штрихи качества
- Добавить раздел *Reproducibility checklist* с версиями Python и базовыми командами:
  ```bash
  conda env create -f environment.yml
  pytest -q
  jupyter nbconvert --execute notebooks/01_eda.ipynb
  ```
- Указать источник данных UCI (ссылка) и объяснить, что файлы уже лежат в `data/raw/`.
- Добавить пример вызова инференса:
  ```bash
  python -m src.models.infer --model data/models/model_best.joblib --input data/raw/adult_sample.csv --output preds.csv
  ```

### E. Синхронизировать документацию и README
- Проверить, что описанные шаги соответствуют текущему состоянию проекта.
- Удалить устаревшие упоминания об отсутствии CI и тестов.

### F. Тесты на «чистой» среде
- Убедиться, что `pytest -q` проходит на свежем окружении без GPU и X-сервера.
- Проверить, что зависимости из `requirements.txt` и `environment.yml` воспроизводимы.
- Зафиксировать минимальные версии библиотек, если тесты не проходят на старых.

## Результат выполнения
После реализации пунктов A–F проект можно считать полностью готовым к публикации как *Census Income Classifier v03*.
